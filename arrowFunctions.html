Arrow Functions

Arrow functions allow you to create functions with a more functional and concise syntax

It allows you to show that a function does one thing and promotes smaller functions.

Beware of lexical `this` when using arrow functions See here for more details

You use the => syntax for arrow functions
You can try the following commands:

                function theFunctionWay(myParameter) {
                    console.log(myParameter)
                }
                
                theFunctionWay("Hello, World")
                

                function theFunctionWayMultipleParams(myParameter, myParameter2) {
                    console.log(myParameter)
                    console.log(myParameter2)
                }
                
                theFunctionWay("Hello", " World")
                
                

                const theArrowFunctionWay = myParameter =>  console.log(myParameter)

                theArrowFunctionWay("Hello, World")
                

                const theArrowFunctionWayMultipleParams = (myParameter, myParameter2) => {
                    console.log(myParameter)
                    console.log(myParameter2)
                }
                     
                theArrowFunctionWayMultipleParams("Hello", " World")
